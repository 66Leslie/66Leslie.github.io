# 66Leslie.github.io

Personal homepage for **Xuewen Zhao (赵学文)**.

- Homepage: https://66leslie.github.io/
- Repository: https://github.com/66Leslie/66Leslie.github.io

## 本地预览 / Local Preview

需要安装 Ruby + Bundler。

```bash
bundle install
bundle exec jekyll serve --livereload
```

浏览器打开 http://127.0.0.1:4000

## 编辑内容 / Content

- 站点配置：`_config.yml`
- 首页内容（分段）：`_pages/about.md` 与 `_pages/about/*.md`
